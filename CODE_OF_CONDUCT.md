# Code Of Conduct

本项目采用 Contributor Covenant 风格的社区行为准则。

## 我们的承诺

为了营造开放、友好与尊重的协作环境，所有贡献者、维护者与参与者都应：

- 尊重不同背景与观点
- 给予建设性反馈
- 对他人的时间与劳动保持基本礼貌
- 以项目目标为先，避免人身攻击和情绪化沟通

## 不可接受的行为

- 人身攻击、骚扰、侮辱性言论
- 歧视性言论或行为（包括但不限于性别、种族、宗教、年龄等）
- 恶意刷屏、挑衅、公开羞辱他人
- 未经许可泄露他人隐私信息

## 处理机制

维护者有权对违规行为进行提醒、警告、临时或永久禁言/封禁。

## 联系方式

如遇到行为问题，请通过仓库维护渠道联系维护者，或使用私密安全报告通道。
